margin:autothe margin-top and margin-bottom values are set to 0 when auto is used.

Recall using trbl on an absolute position element declares the distance between the child's margin and its parent's edge. As an example, using top:0 on a absolute position element means "I want there to be 0 distance away from the top margin of the child to the top edge of the parent (In other words, the top margin of the child is allowed to be as close to the top edge of the parent as possible)i". If we apply rbl=0 as well, that allows the child element to be placed anywhere within the parent because there is no restriction that says "This child element must be 'x' distance away from the parent's edge".
tldr: trbl sets new boundaries within the parent element that the child will position in.

Margin:auto divides the available vertical and horizontal space evenly between its two opposite sides. (Ex: If there is a parent with height 100px, and a child element with height 50px. There is 50px of free space available. Therefore, a margin-top:25px and a margin-bottom:25px will be set). Because using trbl=0, the available vertical and horizontal space is respectively the height and width of the parent element. Therefore, dividing the space (after accounting for the space the child takes up) evenly, using margin:0 will center the child element within the parent.

Example of precedence: top over bottom, left / right precedence depends on if text is left-to-right or vice versa. See precedence by setting top= >0. Note that if top and bottom has a set value, then the bottom value is ignored (ex: top:0 bottom:50px, element is same as top/bot:0)
